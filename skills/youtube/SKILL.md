# YouTube - Video Content Creation

Research YouTube and create video content packages.

## When to Use

Trigger on:
- "youtube video about"
- "create video content"
- "video script for"
- "youtube shorts"
- "video ideas"

## Workflow

1. **Research**: Analyze top YouTube content on topic
2. **Long-form**: Title, description, script outline
3. **Shorts**: 3-5 short-form scripts (30-60s)
4. **Thumbnails**: Visual concepts
5. **Save**: Notion page with all content

## Content Types

- long: Full video content
- short: YouTube Shorts only
- both: All formats (default)
