# Content Pipeline

Full content creation workflow: Research → Article → Social Media → Storage.

## When to Use

Trigger on:
- "create content about [topic]"
- "run pipeline for [topic]"
- "research and write about [topic]"
- "generate social content"
- "content for [topic]"

## Phases

### Phase 1: Research
- Web search for topic
- YouTube content analysis
- Twitter/X trends
- Compile key findings

### Phase 2: Article
- Generate outline
- Write 1500-2500 word article
- Create Notion page

### Phase 3: Social Media
- **Instagram**: 8-10 slide carousel + caption
- **Twitter/X**: 8-12 tweet thread
- **YouTube**: Title, description, script outline
- **Short-form**: 3-5 scripts each for Shorts/TikTok/Reels

### Phase 4: Storage
- Save all to Notion page
- Create Airtable tracking record
- Set status to "Draft"

## Output Platforms

1. Instagram (carousel + reels)
2. Twitter/X (thread)
3. YouTube (long-form + shorts)
4. TikTok (short videos)

## Requirements

- Notion MCP configured
- Airtable MCP configured
- Web search capability
