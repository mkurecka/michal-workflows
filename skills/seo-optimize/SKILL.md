# SEO Optimize - Search Optimization

Improve project SEO with meta tags, structured data, and sitemaps.

## When to Use

Trigger on:
- "improve seo"
- "add meta tags"
- "structured data"
- "generate sitemap"
- "seo optimization"

## Workflow

1. **Analyze**: Current SEO state
2. **Meta**: Titles, descriptions, OG tags
3. **Structured**: JSON-LD schema
4. **Technical**: Sitemap, robots.txt, canonicals
5. **Report**: Changes and recommendations

## Scope Options

- full: Complete SEO audit
- meta: Just meta tags
- structured: JSON-LD data
- sitemap: Sitemap generation
