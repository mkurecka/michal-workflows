# Airtable Fix - Error Processing

Find and fix Airtable records with error status.

## When to Use

Trigger on:
- "check airtable errors"
- "fix airtable records"
- "retry failed records"
- "process errors"
- "airtable status error"

## Workflow

1. **Find**: Query records with Error/Failed status
2. **Display**: Show error details
3. **Process**: List, retry, or auto-fix
4. **Report**: Summary of actions taken

## Actions

- list: Show all errors
- retry: Re-trigger processing
- fix: Auto-fix where possible
