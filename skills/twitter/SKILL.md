# Twitter - Thread & Tweet Creation

Create engaging Twitter/X content for engagement.

## When to Use

Trigger on:
- "twitter thread"
- "create thread"
- "tweet about"
- "x post"
- "twitter content"

## Workflow

1. **Thread**: 8-12 tweet thread with hook and CTA
2. **Standalone**: 5 individual tweets
3. **Series**: Multi-day tweet schedule
4. **Hashtags**: Relevant hashtag suggestions
5. **Save**: Notion page with all content

## Content Types

- thread: Full thread (default)
- tweet: Standalone tweets
- series: Multi-day content
